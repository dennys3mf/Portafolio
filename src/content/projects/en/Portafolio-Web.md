## 🎨 About the Portfolio

This personal portfolio is a digital presentation showcasing my work, technical skills, and professional trajectory as a developer. Designed with a focus on performance and user experience.

## ✨ Features

### 🚀 Ultra-fast Performance
- Built with Astro.js for maximum speed
- Perfect Lighthouse score
- Automatic image optimization

### 🌙 Dark/Light Theme
- Theme switching with local persistence
- Smooth transitions between themes
- Detects system preference

### 🌐 Multilingual
- Support for Spanish and English
- Language switching without page reload
- Localized URLs

### 📱 Responsive Design
- Adapted to all devices
- Mobile-first approach
- Optimized navigation

## 🛠️ Technologies

- **Framework**: Astro.js 5.x
- **Styles**: Tailwind CSS 4.x
- **Interactivity**: React islands
- **Types**: TypeScript
- **Hosting**: Vercel

## 📊 Sections

- Featured projects with filters
- Technical blog with Content Collections
- About me page with timeline
- Comments system with Astro DB
- Contact form

## 🎯 Achieved Goals

- Load speed < 1 second
- SEO optimized for maximum visibility
- WCAG 2.1 level AA accessibility
- Smooth user experience
