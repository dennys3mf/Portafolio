## 🎮 Sobre el Proyecto

CSS Game es una plataforma educativa gamificada que transforma el aprendizaje de CSS en una aventura interactiva. Los usuarios resuelven desafíos de estilización mientras avanzan por niveles de dificultad creciente.

## 🌟 Características

### 🎯 Niveles Progresivos
- 50+ desafíos organizados por dificultad
- Desde selectores básicos hasta Grid y Flexbox avanzado
- Sistema de puntos y logros

### 📝 Editor en Vivo
- Escribe CSS y ve los resultados instantáneamente
- Sintaxis highlighting y autocompletado
- Pistas y soluciones disponibles

### 🏆 Sistema de Logros
- Desbloquea insignias especiales
- Tabla de clasificación global
- Estadísticas de progreso personalizadas

## 🛠️ Stack Tecnológico

- **Frontend**: Astro.js + JavaScript vanilla
- **Estilos**: CSS3 puro con animaciones
- **Editor**: Monaco Editor integrado
- **Despliegue**: Netlify con CI/CD

## 📚 Aprendizaje

Cubre temas como:
- Selectores CSS
- Box Model
- Flexbox y Grid
- Animaciones y transiciones
- Responsive Design
- Variables CSS

## 🎯 Impacto Educativo

Más de 500 estudiantes han usado la plataforma para mejorar sus habilidades en CSS, con una tasa de finalización del 78% y valoración promedio de 4.8/5.
