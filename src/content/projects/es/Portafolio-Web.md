## 🎨 Sobre el Portafolio

Este portafolio personal es una carta de presentación digital que muestra mi trabajo, habilidades técnicas y trayectoria profesional como desarrollador. Diseñado con un enfoque en rendimiento y experiencia de usuario.

## ✨ Características

### 🚀 Rendimiento Ultra-rápido
- Construido con Astro.js para máxima velocidad
- Puntuación perfecta en Lighthouse
- Optimización de imágenes automática

### 🌙 Tema Oscuro/Claro
- Cambio de tema con persistencia local
- Transiciones suaves entre temas
- Detecta preferencia del sistema

### 🌐 Multiidioma
- Soporte para español e inglés
- Cambio de idioma sin recargar página
- URLs localizadas

### 📱 Diseño Responsivo
- Adaptado a todos los dispositivos
- Mobile-first approach
- Navegación optimizada

## 🛠️ Tecnologías

- **Framework**: Astro.js 5.x
- **Estilos**: Tailwind CSS 4.x
- **Interactividad**: React islands
- **Tipos**: TypeScript
- **Hosting**: Vercel

## 📊 Secciones

- Proyectos destacados con filtros
- Blog técnico con Content Collections
- Página sobre mí con timeline
- Sistema de comentarios con Astro DB
- Formulario de contacto

## 🎯 Objetivos Alcanzados

- Velocidad de carga < 1 segundo
- SEO optimizado para máxima visibilidad
- Accesibilidad WCAG 2.1 nivel AA
- Experiencia de usuario fluida
