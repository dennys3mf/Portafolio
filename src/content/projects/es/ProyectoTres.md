## 📱 Descripción del Proyecto

Una aplicación móvil completa de comercio electrónico desarrollada con Flutter, ofreciendo una experiencia de compra fluida tanto en iOS como Android. Integración completa con pasarelas de pago y sistema de gestión de inventario en tiempo real.

## 🌟 Características Principales

### 🛒 Carrito de Compras Inteligente
- Gestión de productos con cantidades
- Cálculo automático de totales y descuentos
- Productos relacionados y recomendaciones
- Guardado de carrito entre sesiones

### 💳 Pagos Seguros
- Integración con Stripe
- Múltiples métodos de pago
- Pagos con tarjeta y carteras digitales
- Historial de transacciones

### 📦 Tracking de Pedidos
- Seguimiento en tiempo real
- Notificaciones push de estado
- Historial completo de pedidos
- Sistema de devoluciones

### 👤 Perfil de Usuario
- Gestión de direcciones de envío
- Métodos de pago guardados
- Wishlist de productos
- Valoraciones y reseñas

## 🛠️ Stack Tecnológico

- **Framework**: Flutter 3.x
- **Lenguaje**: Dart
- **Backend**: Firebase (Firestore, Auth, Cloud Functions)
- **Pagos**: Stripe API
- **Notificaciones**: Firebase Cloud Messaging
- **Estado**: Provider + Riverpod
- **Testing**: Unit tests + Widget tests

## 📊 Métricas

- 10,000+ descargas en primera semana
- Calificación promedio 4.6/5
- Tiempo de carga < 2 segundos
- 95% de transacciones exitosas

## 🎯 Logros Técnicos

- Arquitectura limpia con separación de capas
- Manejo eficiente de estado global
- Optimización de imágenes y caché
- Soporte offline-first
- Animaciones fluidas 60fps
